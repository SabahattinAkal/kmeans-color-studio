# Changelog

## 1.2.0 — 2026-07-17

- Consolidated the useful parts of seven historical Lena exercises into tested diagnostics.
- Added true grayscale conversion, histogram equalization, RGB channel histograms, and bit-depth reduction.
- Added `--diagnostics` artifacts with redistribution-safe input images.

## 1.1.0 — 2026-07-17

- Added CIELAB quantization for perceptually improved clustering.
- Added automatic palette-size analysis with elbow JSON/SVG reports.
- Added color-space selection to the CLI and API.
- Added public-project governance, security guidance, and MIT licensing.

## 1.0.0 — 2026-07-17

- Rebuilt the original 2022 exercise as an installable, tested CLI and web application.
